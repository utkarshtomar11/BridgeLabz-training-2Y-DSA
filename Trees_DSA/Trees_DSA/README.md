## 🧠 Key Takeaways

- **Inorder** traversal of a BST gives nodes in **sorted ascending order**
- **Preorder** is used for **copying/backup** of tree structure
- **Postorder** is used for **deletion** and **directory size** calculation
- BST operations are **O(log n)** for balanced trees and **O(n)** for skewed trees
- Expression trees map directly to **infix / prefix / postfix** notations
